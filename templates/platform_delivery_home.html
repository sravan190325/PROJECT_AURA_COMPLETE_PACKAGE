Frequently Asked Questions

Do I need API credentials to try the demo?

No! The demo uses a special token (demo) that works without any account setup. Perfect for exploring the feature.

What happens when I use real credentials?

Real projects are created on your actual SmartSheet or Jira accounts. Excel files are generated and downloaded to your computer. Everything is automated—no manual setup required.

Can I create projects on multiple platforms?

Currently, you can select one platform per deployment. You can run the process multiple times to create the same project on different platforms if needed.

How long does it take to create a project?

Excel: <5 seconds | SmartSheet: 15-30 seconds | Jira: 30-60 seconds. Times vary based on project complexity and network speed.
