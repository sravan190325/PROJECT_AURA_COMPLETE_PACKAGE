## 🎨 Color Guide

### Status Indicators

🟢 **Green** (#70AD47) – Positive, On-track, Success  
🟡 **Amber** (#FFC000) – Caution, In-progress, Warning  
🔴 **Red** (#C5504F) – Risk, Issue, Delayed  
⚪ **Gray** (#BFBFBF) – Planned, Neutral, Baseline  

### Health Status

🟢 **GREEN - Low Risk** – Confident, well-managed project  
🟡 **AMBER - Medium Risk** – Some concerns, mitigation in place  
🔴 **RED - High Risk** – Significant risks, active management needed

## 💡 Tips & Tricks

### Navigation Shortcuts
- Use **Ctrl+Home** to go to first sheet
- Use **Ctrl+Page Down/Up** to switch sheets
- Click sheet tabs at bottom to jump

### Filtering Data
- Click the funnel icon (⊥) in header row
- Select values to show/hide
- Great for focusing on specific items

### Sorting Data
- Click column header dropdown
- Sort A-Z or Z-A
- Sort by date or value

### Frozen Headers
- Headers stay visible when scrolling
- Easier to understand data
- Works on all main sheets

### Print
- Use **File → Print**
- Recommend landscape orientation
- Each sheet prints separately

## 🤔 FAQ

**Q: Why is the workbook different?**  
A: We consolidated 21 sheets into 12 focused sheets to reduce duplication and improve clarity.

**Q: Can I still use the old version?**  
A: Yes! Use `?generator=enhanced` in the URL to get the 21-sheet version.

**Q: Do I need to update anything?**  
A: No! Just generate and download as usual. The workbook format changed, but the process is the same.

**Q: Where is information I used to see?**  
A: Check the sheet consolidation guide:
- Project Charter → merged to Project Details
- Assumptions → moved to RAID Register
- Risk Register → merged to RAID Register
- Staffing Plan → consolidated to Resource Plan
- Leave Planner → part of Resource Plan
- Project Plan → improved as Detailed Project Plan

**Q: How do I generate the old format?**  
A: Add `?generator=enhanced` to the generation URL, or `?generator=standard` for the original 10 sheets.

**Q: Is everything still in the workbook?**  
A: Yes! All information is consolidated, not removed. It's just better organized.

**Q: Can I customize the colors?**  
A: Contact your IT/Development team. Color customization requires code updates.

**Q: When should I use which sheet?**  
A: Start with Home (00_Home), then go to Dashboard (01). Refer to the Sheet Guide above for details.

1. Open Home page (00_Home)
2. Check Executive Dashboard (01_Executive_Dashboard)
3. Review AI Summary (02_AI_Project_Summary)

3. Review Detailed Plan (05_Detailed_Project_Plan)

1. Check Dashboard health (01_Executive_Dashboard)
2. Review RAID Register (09_RAID_Register)

1. Open Weekly Status (11_Weekly_Status)

1. Check Milestone Tracker (07_Milestone_Tracker)
2. Review upcoming milestones
